# sorting-algorithm-analysis
Implemented and benchmarked Bubble Sort, Quick Sort, and Merge Sort in Python for CSCI 3320 Data Structures at the University of Nebraska Omaha.

**What it does:** Generates random and pre-sorted datasets of up to 500,000 elements, measures execution time for each algorithm, and visualizes performance comparisons using matplotlib.

**Languages/Libraries:** Python, matplotlib

**Key finding:** Quick Sort and Merge Sort significantly outperform Bubble Sort on large datasets.
